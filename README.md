# BIT Website 

A simple website project showcasing basic web pages and features.

## Project Overview

This is a mini website for **Bannari Amman Institute of Technology (BIT)**. It includes:

- Home page
- About page
- Departments page
- Contact page
- Vision & Mission page
- Responsive design with Bootstrap 5
- Docker support for easy deployment


## Features

- Modern responsive design using Bootstrap
- Navigation bar with dropdown
- Image carousel on home page
- Department cards with hover effect
- Footer with contact information
- Dockerfile for containerized deployment

## Tech Stack
## Frontend

- HTML5 – Structure of web pages
- CSS3 – Styling and layout
- Bootstrap 5 – Responsive design and components

## Backend / Server
- Nginx – Web server for serving static files (via Docker container)

## Containerization

- Docker – Containerize the website for easy deployment

## Version Control

- Git – Local version control
- GitHub – Remote repository and collaboration

## Optional / Tools

- VS Code – Development
- Browser (Chrome/Edge/Firefox) – Testing
